# HouseOfDev
               HOUSE OF DEV
                         proyecto de plataforma5


Descripcion el proyecto:
Crear una aplicación web, para una inmobiliaria, que tiene como objetivo el de poder agendar
una visita a las viviendas.

Stack de tecnologías involucradas:
-NodeJS
-Nodemon
-Express
-Sequelize
-Postgres
-Volleyball
-Bcrypt
-JWT
-Nodemailer
